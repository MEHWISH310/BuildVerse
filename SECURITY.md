# Security Policy

## Supported Versions

Currently, only the latest version of BuildVerse is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

Security is a top priority for us. If you discover a security vulnerability in BuildVerse, please **do not open a public issue**. 

Instead, please report it privately to ensure it can be addressed safely before being made public.

To report a vulnerability:
1. Go to the **Security** tab of this repository.
2. Under "Reporting", click on **Advisories**.
3. Click the **Report a vulnerability** button.
4. Provide a descriptive title and a detailed description of the vulnerability, including steps to reproduce it.

You should receive a response within 48 hours. If the vulnerability is accepted, we will work with you to patch it and publish a security advisory. We greatly appreciate your efforts to responsibly disclose your findings and keep the BuildVerse community safe!
